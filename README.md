# PlayStation Perspective (PSP XMB Interface Emulator)

A high-fidelity web-based emulator of the PlayStation Portable (PSP) XMB (XrossMediaBar) interface, featuring several mini-games and dynamic themes.

## Project Structure

This project follows a professional web development directory structure:

- `src/`: Contains all source code.
  - `css/`: Stylesheets (`boot.css`, etc.)
  - `js/`: JavaScript logic (`games.js`, `boot.js`, etc.)
- `assets/`: Static assets.
  - `img/`: Images (`psp-vector-logo.png`, etc.)
- `docs/`: Historical documentation or initial drafts.
- `misc/`: Temporary files or miscellaneous artifacts.
- `index.html`: The main entry point for the application.

## Mini-Games Included

- **Snake**: Classic snake game.
- **Tower Stacker**: Stack blocks to build the tallest tower.
- **Pinball**: A physics-based pinball experience.
- **Click Speed**: Test how fast you can click.
- **Typing Test**: Challenge your words-per-minute.
- **Simon Says**: Test your memory with light patterns.

## Getting Started

Simply open `index.html` in any modern web browser to start the PSP experience.

### Key Controls
- **Up/Down/Left/Right Arrows (or WASD)**: Navigate the XMB.
- **Enter / X (s key)**: Confirm / Start game.
- **Backspace / O (d key)**: Back / Cancel.
